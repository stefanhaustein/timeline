Timeline

This project pulls timelines from Wikipedia, parses them and puts them into a coherent zoomable view. This is mostly inspired by Huang's great Flash app "The Scale of the Universe" and George Santayana's famous quote.

I have tested this only in Chrome so far.

Use the scroll wheel togehter with the shift key to zoom in or out. '+', '-' and the cursor keys should also work.

"Ma" stands for "Million years ago".

The source code is available at github.com/stefanhaustein/timeline.
